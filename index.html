Hackathon projects

  • Similaripy: A clustering algorithm to find similar texts given a vector representation of the texts. WINNER
  • VOLVO Share: Easy-to-use car pooling app. WINNER
  • Donework: Content generator given a table of contents using Deep Learning and a Search similarity engine.
  • HealthCare: Communication app for doctors and nurses. RUNNER-UP
  • NeverRotten: Organize your food and never let it get rotten. WINNER
  • WeatherPlayer: Music mixer using weather data built in Junction'18. RUNNER-UP
  • SOMA: a webapp will show you only the daily news that you want to see: the positive ones. WINNER

Personal projects

  • CaffeParserPrototxt: This tool is able to parse a Caffe deploy prototxt to a train prototxt and vice versa..
  • Yello: This repository contains a project that combines DJI Tello and Deep Learning's framework Darknet using the Tiny Yolo architecture.
